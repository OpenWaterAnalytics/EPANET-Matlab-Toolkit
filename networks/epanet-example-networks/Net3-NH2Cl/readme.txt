Monochloramine Decomposition in EPANET Example Network Net3
===========================================================

This example models monochloramine decomposition in the
standard EPANET example network Net3. The files included are:

  net3.inp       --   the EPANET network input file set up to compute
                      average conditions over the last 48 hours of a
                      360 hour simulation
  net3-nh2cl.msx  --  the MSX reaction file
  net3-nh2cl-[gc|vc].msx -- the MSX reaction file with the
                      compiler option turned on.
  net3-nh2cl.rpt  --  the report file containing the MSX results
